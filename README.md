# LeoBot

## Use TradingView charts to trade on Binance

LeoBot uses TradingView charts to send trade signals to an Azure Function that places orders on Binance Futures exchange.

---

### Prerequisites

- TradingView Pro
- VSCode
- Azure Functions
- NodeJS
